Take payments on CredtCard, Deposit and Boleto  using LatamGateway.

== Description ==

The **Latam Gateway for WooCommerce** plugin allows retailers to accept payments via **CredtCard, Deposit and Boleto** using the Latam Gateway API. By completing a purchase users are redirected to the Latam Gateway platform.

** Latam Gateway ** works with the leading banks and card brands in the market, so your customers have the autonomy to choose the best way to pay.

**Banks available for Deposit:**

- Bradesco
- Itaú
- Banco do Brasil
- Santander
- Caixa Econômica Federal

**Card brands availables:**

- Visa
- MasterCard  
- American Express 
- Discover  

== Installation ==

1. Access the WordPress admin panel.
2. In the right sidebar, go to: Plugins> Add New.
3. In the search field enter "Latam Gateway for WooCommerce". 
4. Select our plugin from the list and click "Install Now".
5. Finally, click the "Activate" button after installation.

= Manual Installation =

The manual installation method involves downloading our plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here] (http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

== Requirements ==

- [Latam Gateway](latamgateway.com/parceiro/acesso) account.
- API account key.
- Plugin "WooCommerce" activated and installed.
- Plugin "Brazilian Market on WooCommerce" activated and installed.
- PHP 5.6 minimum version
